<h2>Developer Guide</h2>
<p>
<h3>1. Prerequisites:</h3>
<ul>
<li>NodeJS</li>
<li>NPM</li>
<li>Solidity</li>
<li>Ganache - Truffle Suit</li>
</ul>
</p>


<h1></h1>

<p>
<h3>2. Install Packages:</h3>

`$ npm install`

</p>

<h1></h1>


<p>
<h3>3. Truffle Commands:</h3>

+ Complie contracts         

  `$ truffle complie`

+ Migrate Contracts to Ganache   

  `$ truffle migrate`

+ Test contracts 

  `$ truffle test`
